Projet ISN - Tic-Tac-Toe

Bienvenue sur le site web de notre projet ISN 2012-2013. Cette page a pour but une présentation efficace de notre projet. Le projet à pour but de créer le jeu du tic-tac-toe ou morpion en Python. Le but du jeu est est d'aligner 3 marques, ici un rond ou une croix pour former une ligne. Chaque joueur joue à tour de rôle. Le jeu se déroule sur un plateau composé de 3x3 cases, 3 lignes et 3 colonnes. Celui qui arrive à aligner trois de ses marques, en ligne, en colonne en diagonale a gagné. Si aucun n'y arrive, le jeu se finit sur un match nul. Vous pouvez voir le projet en cliquant en haut à gauche ou récupérer le projet en cliquant sur Download en haut à droite. Vous pouvez aussi récupérer le projet par SSH :

$ git@github.com:gaelfoppolo/tic-tac-toe.git

Si vous utilisez GitHub sur Mac, vous pouvez simplement synchroniser votre dépôt et vous pourrez voir les nouvelles branches du projet.
